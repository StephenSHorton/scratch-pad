Scratch Pad

Desktop notes for Claude Code

Floating sticky notes that Claude can read and write. Leave yourself context between sessions. See your plans on screen.

Download for macOS View on GitHub

macOS 10.15+ · Apple Silicon & Intel · Free & open source

Floating Notes

Borderless sticky notes that float on your desktop. Always visible while you work. Drag them anywhere.

Markdown Support

Full markdown rendering — headers, code blocks, lists, bold, italic. Double-click to edit any note.

Claude Integration

Built-in MCP server lets Claude Code create, read, and update your notes directly. No copy-pasting.

Session Handoff

Claude can leave plans, summaries, and context for your next session. Pick up right where you left off.

How it works

1

Install the app

Download the DMG, drag to Applications, and open Scratch Pad. It lives in your menu bar.

2

Connect Claude Code

Run one command to register the MCP server:

claude mcp add scratch-pad -- \
  "/Applications/Scratch Pad.app/Contents/MacOS/scratch-pad-mcp"
3

Start using it

Ask Claude to create a note, leave a plan, or summarize your session. Notes appear instantly on your desktop.

MCP Tools

Five tools Claude Code can use to manage your notes.

note_create

Create a new note with a title, markdown body, and color

note_list

List all active notes

note_read

Read a specific note's content

note_update

Update a note's title, body, or color

note_clear

Delete one or all notes

Installation Notes

First launch on macOS: The app is not yet signed with an Apple Developer certificate. After installing, you may need to remove the quarantine flag:

xattr -cr /Applications/Scratch\ Pad.app

Then open the app normally from Applications.
